# Personal-Expense-Tracker
A Command Line tool to Track personal Expenses.

Verison 1.0 Features:
1:Adding Expenses With Date And Category.
2:Finding Expenses Using Date And Category.
3:Find Which Category Has Highes Expenses.

Version 2.0 New Features:
1:Adding Multiple Expenses ( Not Fully Dynamic)
2:Finding Total Expenses Using Dates ,Category and Over Expenses
3:Some other improvements
