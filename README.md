# Personal-Expense-Tracker
A Command Line tool to Track personal Expenses.

Verison 1.0 Features:
Adding Expenses With Date And Category,
Finding Expenses Using Date And Category,
Find Which Category Has Highes Expenses.

Version 2.0 New Features:
Adding Multiple Expenses ( Not Fully Dynamic),
Finding Total Expenses Using Dates or Category and Overall Expenses,
Some other improvements
